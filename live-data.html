How new pages should be added

Keep every new page useful and unique. Do not publish pages that only repeat the same counter with a different title.

1. Unique page angle

Each page should have its own search intent, H1, introduction, estimate and explanation.

2. Internal links

Every detail page should link to the homepage, this Live Data hub and 4-8 related pages.

3. Method and sources

Add a short method note explaining UTC-based estimates and link to the data sources page.
